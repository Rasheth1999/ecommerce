## REACT ECOMMERCE WEBSITE

- FrontEnd using language: ReactJS
- BackEnd using language: NodeJS
- BackEnd Database: XAMPP Server(MYSQL)

First run the server

- cd server
- npm i
- npm start

Second run the client or admin page Client page to order the product

- cd client
- npm i
- npm start
- visit localhost:3003/

Admin page to create a new product, update a product, view order list

- cd admin
- npm i
- npm start
- visit localhost:3002/
